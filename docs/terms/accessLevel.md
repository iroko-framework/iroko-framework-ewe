# accessLevel

**IRI:**  
`https://iroko-framework.github.io/iroko-framework-ewe/terms/accessLevel`

## Definition
Governance designation indicating the disclosure level for culturally sensitive fields.

## Expected values (controlled)
- `open` — general botanical and bibliographic description
- `limited` — high-level contextual description without procedural detail
- `restricted` — culturally sensitive context; non-operational and not published beyond approved contexts

## Notes
This term controls disclosure for fields such as ritual context, associated spirits, and other culturally governed knowledge.
