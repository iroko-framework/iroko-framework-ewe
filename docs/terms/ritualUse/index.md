# Ritual Use Scheme

**IRI:**  
https://iroko-framework.github.io/iroko-ewe-module/terms/ritualUse

**Type:**  
skos:ConceptScheme

**Preferred label:**  
Ritual Use Scheme

---

## Definition

Controlled values for high-level ritual classification associated with ethnobotanical records. This scheme supports non-operational description and separates public botanical description from culturally governed ritual specificity.

---

## Scope note

This scheme is intended for classification only. Procedural detail is not included. Disclosure and circulation of ritual context should be governed by the associated accessLevel.

---

## Concepts in this scheme

- Purification and Cleansing
- Protection and Boundary Setting
- Invocation and Communication
- Healing and Restoration
- Offering and Devotion
- Rites of Transition
- Cosmological or Symbolic Association
